Sobre o Telemob

O Telemob foi criado para ajudar o cidadão a pressionar os deputados federais e senadores do seu estado a votarem conforme a vontade de seus eleitores, resistindo a lobbies contrários aos interesses da sociedade.

Como funciona

O Telemob oferece 3 coisas:

Priorização dos parlamentares a serem contatados
No passo 2, o primeiro parlamenar da lista é sempre o que recebeu menos contatos até o momento, em cada estado. O objetivo é distribuir a pressão popular entre todos os representantes, evitando que somente os parlamentares mais famosos sejam pressionados. Os parlamentares são ordenados de acordo com o número de contatos reportados através deste site, em ordem crescente.

Dados de contato dos parlamentares
O Telemob informa os dados de contato dos parlamentares: fone do gabinete, fax, e-mail e endereço postal para envio de telegrama. E também o link para a página de envio de telegrama do site dos Correios.

Registro do resultado do contato
O Telemob não envia nenhuma mensagem, e não acreditamos no uso de ferramentas de spam como forma de pressão. O cidadão deve escolher uma forma de contato, enviar a mensagem ele mesmo, e depois voltar à página e indicar de que forma mandou a mensagem e se teve sucesso.

Privacidade

Nenhuma informação pessoal dos usuários é armazenada no sistema. Os registros de contatos realizados são anônimos.

Créditos

O sistema Telemob foi desenvolvido por um pequeno grupo de voluntários entre 14 e 16 de março de 2014.

Este projeto é software livre (GPL v.3), e foi desenvolvido em Python com Django. O código está no Github: https://github.com/garoa/telemob. Faça bom uso dele!

Contato

Para sugerir melhorias ou reportar falhas, cadastre uma pendência no repositório do projeto ou escreva para telemob@turing.com.br.
